# Branch — AutoCAD Plugin Releases

This repository hosts installer releases for **Branch**, Leaf Automation's drafting automation plugin for AutoCAD.

Branch automates solar stringing, homerun routing, and circuit tagging directly inside AutoCAD — so your team spends less time on repetitive drafting and more time on engineering work that requires judgment.

## Download

Installers are distributed through the [Leaf Automation dashboard](https://www.leafautomation.ai/dashboard) after account creation and EULA acceptance.

**Supported platforms:**
- AutoCAD 2018–2026 on Windows (including Architecture, MEP, and Electrical editions)
- AutoCAD LT is not supported

## Installation

1. Close AutoCAD completely before running the installer
2. Run `BranchInstaller_18-26.msi`
3. Open AutoCAD — Branch commands will be available immediately
